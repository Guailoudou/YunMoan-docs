# 小游戏群组-三周目

## 介绍

一个**提供**诸多资源**支持**群友、作者、UP主等牵头组织人**发起**不定时不限人的**小游戏**平台。

## 适宜人群

- 喜欢参与多人游戏的朋友
- 想和朋友欢乐的朋友
- 想测试自己地图的作者
- 想发起但没有服务器、没有地图的朋友

## Q&A

```markdown
Q:会不会管理霸权
A:不会
- 你可以收集好相关证据直接私聊群主举报。
- 群友由管理审判，管理由群主审判。公开公正。
- 群主？群主不参与游戏，只认证据。
```

```markdown
Q:我只是一个普通群友如何发起/使用小游戏
A:随便在群里@相关管理，说明得当
- 时间：需要具体到某天几点到几点
- 游戏：如果已经录入面板，直接说名字即可。如果没录入面板需要提供地图文件及来源
- 地址：群号、地点、服务器地址
- 公私：允许私开，但要提前说明，不然默认群内公开，视情况推发到其他同类型群聊。
- - 支持生日会、朋友小聚、企划录制等场景私开
```

```markdown
Q:能不能给我地图文件
A:不能，有二次分发转载嫌疑，保证作者权益。不会直接提供文件。
- 收录文档内存在 官方地址、汉化来源 还请支持给大家做饭的人
- 提供技术指导，哪怕你要复刻一个我们
```

```markdown
Q:我不认这个审判
A:多数情况既然能到群内发起并公开审判的时候，是给过机会和劝导的
- 本群遵守社会主义核心价值观：富强、民主、文明、和谐，自由、平等、公正、法治，爱国、敬业、诚信、友善
- 本群没有村规，仅有保证氛围和谐，确保游戏环境的公约
- 倒也不用太担心，仅在本群联动范围内联黑，圈子有时也很大不是么
- 本群没有公主没有少爷，都是普通人聚在一起游玩而已，自寻他处
```

```markdown
Q:我没有正版能不能玩
A:因存在其他渠道在此发布游戏活动，视情况而定
- 其他UP来此宣传：会提供黑名单，验证与否看UP需求
- 本群发起活动，会在前一天投票相关事项，其中包括验证
- 可以参与其他使用皮肤站的渠道
- 本群在一些大型节假日活动时会群内抽正版
```

```markdown
Q:我想玩XX地图但群组未收录时
A:
- 提供地图来源(尽可能官方)给相关管理
- 公：会录入群组写入文档
- 私：游玩结束时会删除
```

```markdown
Q:我想玩XX类型的地图，但群组少或者没有
A:
- 跟管理说明
- 管理发起投票
- 视投票情况决定后续测试录入的优先级
```

```markdown
Q:我没有服务器/穿透卡
A:群组开服，玩家只需要参与即可
```



## 流程规范

- 时间

  - 每天均可发起小游戏
  - 9:00-21:00全员@
  - 其他时间静默允许发布但不会全员@

- 宣发，具体可看群内发布模板

  信息完整便于玩家选择

  - MC本体信息：版本 核心 纯净/模组

  - 地图信息：地图名称 地图类型 国内介绍(建议视频或详细文档的链接 避免信息过长)

  - 其他要求：语音 正版/离线/皮肤站验证 人数等

- 没有村规

  - 没有公主没有少爷，都是普通人聚在一起打发时间、同获快乐

## 群组支撑

- 服务器

  - 7950-64G 用于小游戏群组常驻

  - 7950-64G 用于大型地图或其他（盘灵古域、CTM等）

  - 9950X-192G 用于**人多**时的备用选项

    TACZ枪战多人时，客户端比服务端先扛不住

    数据包不逆天时，50-100+

- 网络

  - 主要线路：移动500M上行专线
  - 加速线路：阿里云BGP200M：广州、武汉、上海、北京、深圳
  - 国外线路：香港、英国（不如开加速器到国内用主线路）

- 地图预制

  - 已预制130+地图，覆盖大部分类型小游戏，做到随玩随开
    - 语音
    - 皮肤站验证(待实装，初步准备魔改MCSM做两套启动)
  - 仍有700+地图待测试待汉化待录入
    - 有诸多地图推荐区UP测试
    - 如测试优秀可推送到汉化组(如果他们档期不忙的话)或就地机翻(后续边玩边调整)
    - 录入小游戏会尽量寻找添加视频介绍、游玩实况(UP可以投稿)

- 诸多同道

  公益群体：整合包轮换、小游戏轮换

  汉化群体：测试、汉化、提供

  制作群体：制作、测试、提供

  - 推送机器人：会在群组群发布消息推送到其他加入群聊
  - 插件：群组插件，游戏内开关其他小游戏(待实装)
  - 皮肤站：已部署准备待实装(考虑云黑 以及 mojang验证爆炸 平时不会启用)
