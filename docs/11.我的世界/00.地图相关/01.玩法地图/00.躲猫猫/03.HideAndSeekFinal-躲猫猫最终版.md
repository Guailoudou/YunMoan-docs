# HideAndSeekFinal-躲猫猫最终版

## 基本信息

**版本：**1.20.4

**地址：**[PM](https://www.planetminecraft.com/project/hide-and-seek-1-20-4-3-modes-normal-block-hunt-mob/)

**作者：**[Mynoteet](https://www.planetminecraft.com/member/mynoteet/)

**人数：**2+

## **介绍：**

搜寻者：你必须找到你的对手藏身之处，只需用你的剑击中他们，一击杀死他们，如果对手逃跑，你还可以提高速度来追上他们。
但是，如果您正在玩“Mob”模式，则需要先杀死有问题的“假”Mob，玩家很快就会出现。

隐藏者：隐藏，你有 60 秒的时间隐藏。一旦这些时间过去，研究人员就会在地图中生成，并为你分配一根钓鱼竿。右键单击以更改声音并放下钓鱼竿以播放声音（每个声音之间都有冷却时间）。如果您在 Block Hunt 或 Mob 模式下玩游戏，您将拥有第二根钓鱼竿。右键单击一个方块或生物，将自己变成一个！
