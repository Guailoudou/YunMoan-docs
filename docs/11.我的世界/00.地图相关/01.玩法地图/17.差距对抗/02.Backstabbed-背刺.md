### 开始游戏：

要玩，请下载 zip 文件并将其解压到您的世界保存文件夹中，然后在加入时输入/reload。确保您正在运行Minecraft Java 1.21.4。

加载世界后，您将被放置在电影院中。要开始游戏，请走向左侧的电影海报，右键单击一张以选择地图。然后，前往海报对面的剧院房间，坐下来排队。右键单击引座员村民开始！背刺！适合4-24 名玩家玩。

请务必阅读您库存中的《游戏方法》或电影院主大厅讲台上的《游戏方法》以访问游戏选项，装备解锁的外观装饰，并了解有关游戏的更多信息！

要观看比赛，请使用楼梯或电梯前往电影院二楼，然后在剧院房间的阳台上坐下。

### 游戏选项

从 1.3 版本开始，观众席对面的二楼会显示游戏选项。您可以使用这些选项自定义某些游戏参数，下面是简要概述：

  重置为默认值：将所有游戏设置重置为默认值
  自动开始：绕过引座员，在投票结束时自动开始游戏倒计时。
  游戏时间：延长或缩短游戏时间限制。
  杀手人数：自定义游戏中的杀手人数，介于 1 和 3 之间（含 1 和 3）。请注意，如果只有三名玩家而您将杀手人数设置为 3，游戏将立即结束。此外，还有一个自动选项，为 3-7 名玩家选择 1 名杀手，为 8-17 名玩家选择 2 名杀手，为 18 名以上玩家选择 3 名杀手。
  地图动画：控制漂浮岛上的水车和螺旋桨是否有动画。
  从废料开始：控制玩家是从 0 还是 1 名废料开始。枪手会立即失去他们的废料。
  杀手友军误伤：允许或禁止杀手互相残杀。
  更新玩家数量：启用或禁用更新游戏内boss栏。

选项板上还列出了所有当前地图。您可以切换可用的地图，但是，在地图仍可投票时禁用它并不会立即将其移除。我们下方提供了一个“随机地图”按钮，用于在更改地图后更新地图选择。请注意，如果玩家已开始投票，此按钮将不起作用。此外，

还有一些选项未在选项板上列出，但在书中可用：

  黑暗：在游戏中为玩家提供黑暗效果。
  允许凶手摧毁枪支：应一位开发者的要求，一个旧的bug行为被重新引入，允许凶手使用其刀具召回物品的闪电攻击摧毁地面上的枪支。

游戏规则在游戏运行时可能不会更新。

最后，如果您是服务器管理员，并且希望玩家使用某些预设规则进行游戏，您可以根据自己的喜好自定义这些规则，然后运行/function mcm:lobby/options/lock_option。要重新启用更改选项，请运行/function mcm:lobby/options/unlock_options。
要阻止人们通过书籍更新游戏规则，请运行 / scoreboard players set $restricted GameRules 1，并运行/scoreboard players set $restricted GameRules 0以重新启用此功能。

#### 大厅

钟声：敲响剧院钟声
进入-18 1 53 的咖啡馆，然后前往楼上的阳台。面朝南，跳到-17 6 51 的鲜红色遮阳篷上。打开-18 5 60 和-19 5 59 的深色橡木活板门，跳上去到达-17 6 62 的深色橡木遮阳篷。使用丛林活板门爬上-18 10 62 的灰色地毯。面朝北，跳上砖砌屋顶。再次面朝南，跳到-18 63 61 的丛林活板门上。使用丛林活板门跳上剧院的屋顶。向东走，找到剧院塔楼的红树林入口。完成塔内的梯子跑酷，爬过深色橡木活板门找到钟声。敲响钟声获得成就。

#### 葡萄园

长矛：在葡萄园中找到乌龟长矛
。使用任意入口进入下水道。导航至 1759 76 2136 的长矛并站在附近以获得成就。

远离窥探：解锁葡萄园的主要景点
从花盆中收集花朵等有机材料，并将 1700 93 2040 的堆肥器装满以生产骨粉。手持骨粉，右键点击发光浆果藤蔓的最底部部分，例如 1693 90 2072 处的部分，以生产葡萄。进入下水道并在 1709 74 2059 找到酿酒厂入口。用葡萄右键点击 1690 75 2059 的酿造台以打开它。里面应该有一个酒瓶。拿起酒瓶前往酒窖，酒窖的入口位于1743 79 2084，可从下水道进入。右键点击位于1754 84 2091的酿造台，用酒瓶打开它。将酒瓶放入酿造台UI的瓶槽中。地下酒吧现在应该出现在1710 74 2083。进入并点击点唱机上的按钮即可获得成就。

#### 图书馆

惊悚：唤醒图书馆中的血月。
每进行一次图书馆匹配，地图各处书架的侧面都会生成5本书。找到后，右键点击一本书将其拾取，然后将书放入一楼的书籍归还处。归还2本书将打开地下室入口，归还5本书将启动血月事件。要完成事件，请前往地下室最西边的墙壁，打开下方房间讲台上的书籍，然后点击“继续”。

#### 漂浮岛

飞艇船长：进入船长舱。
在飞艇内部坐标-632 52 -6处轻按控制杆。获取传送水晶（可以通过击打一颗带有粒子的紫水晶获得，紫水晶位于西部岛屿周围），然后站在坐标-636 56 3处的船长舱门前。手持水晶并面向舱门右键，即可传送至舱内并获得成就。

#### 发射场

火箭骑士：在发射场乘坐火箭。
首先，玩家必须找到并按下隐藏在地图各处的 10 个石头按钮。按钮位置如下：
1. 发射台上的汽车内
2. 火箭正下方的水池中
3. 地图西南区域的液氢罐底部
4. 地图东南区域的液氧罐顶部（使用楼梯上去）

您可以使用一些实用的命令来探索比赛之外的地图、禁用特定地图等等。请注意，您需要拥有操作员权限才能使用这些命令。

### 开发者命令指南（剧透！）

探索地图：
要在比赛之外探索地图，您必须首先使用以下命令加入开发团队：
/ function mcm：dev

输入此命令将允许您绕过边界框限制，让您传送到任何地方，还将带您到具有到每个地图的传送点的调试室。

要返回正常游戏，请使用以下命令离开开发团队：
/ team join nametags [玩家名称]
/ gamemode adventure
/ tp 1 1 70（如果您想返回大厅）


随机地图：
要随机播放大厅投票区域中显示的地图，请输入以下命令：
/ function mcm：dev / shuffle_maps


切换地图：
您可以切换特定地图是否可以出现在投票区域中。 可以使用以下命令对每个地图执行此操作：
/ function mcm：dev / toggle / [地图名称]

例如，输入命令/ function mcm：dev / toggle / airship将禁用或启用漂浮岛地图。 请注意，某些地图的内部名称与其显示名称不同。

您可以使用相应的命令启用或禁用所有地图：
/function mcm:dev/toggle/enable_all
/function mcm:dev/toggle/disable_all


创建虚拟玩家：
您可以使用虚拟玩家开始游戏进行测试。为此，请使用以下命令：
/function mcm:dev/fakeplayers。

这将在大厅中装饰装备区旁边生成充当虚拟玩家的盔甲架，让您可以仅与1或2名真实玩家开始比赛。请注意，如果只有1名真实玩家，比赛将立即结束。

要移除虚拟玩家盔甲架，请在大厅中输入以下命令：
/kill @e[type=minecraft:armor_stand,tag=fakeplayer,tag=queued]
