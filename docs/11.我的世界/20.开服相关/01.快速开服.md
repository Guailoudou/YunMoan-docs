# 快速开服

**JAVA**
不带网易玩 基岩不需要 基岩会在互通服场景下 写在**详细开服**
不考虑性能、不考虑优化、不考虑功能（权限组、局内语音、在线地图）
开都开不起来，你还想这么多？先跑起来吧
需同时满足存在**服务端**，**公网服务器**（**家里带宽有公网也可**），一点**电脑知识**

## 服务器
这里不考虑**面板**，你都买面板了，你家客服呢？
（其实因为面板各有差异，难以详尽，后续有机会再写）
这里只考虑**windows**服务器
**linux**会放在**详细开服**那

项目：https://github.com/Griefed/ServerPackCreator

#### 需要寻求社区帮助
或者自己搓一个，搓服务端通用方式会放在**详细开服**那
这里只简单提一嘴
通常**模组端**使用Forge核心，墨端，模组插件混合端放在**详细开服**
把客户端的**config**、**kubjs**、**mods**文件夹拖到安装好的服务端文件夹里
将mods里**仅客户端**模组删除，能启动90%整合包，点名贪婪2、深渊之旅等
模组是否仅客户端是否服务端、客户端都需要，请去MCMOD一个个搜，最朴素也是最有效的办法
别跟我说什么fabric或者别的什么核心有仅客户端模组也能启动，人家可以根据**规范**主动屏蔽
那还有不规范的制作者，它遇到也一样出错

## 过程
### 安装JAVA环境
文件自行寻找 安装的时候一直下一步就行（别担心 装常用的以下四个 也才7 800M）
- 1.20.5 +	
JDK 21
- 1.20 - 1.20.4	
JDK 21（推荐） 或者 JDK 17
- 1.17 - 1.19.4	
JDK17
- 1.13 - 1.16.5	
JDK 11（推荐） 或者 JDK 8
- 1.12.2 -	
JDK 8
### 处理服务端
1.解压出来
1.1运行一下**.bat**文件或者**.ph1**文件
启动不起来检查你的java版本
和服务端的java版本
这里大概率是java不对
右键上面运行的文件 选择 打开方式 选择txt文本
修改 java 为 "C:\Program Files\Java\版本\bin\java.exe"
（参照路径找对应版本的java）
![img](./01.%E5%BF%AB%E9%80%9F%E5%BC%80%E6%9C%8D.assets/m_a76437f59b9fa04fe417c1c7758870e5_r.png)

#### 2.1修改eula文件

里面的false 改成 true

#### 2.2修改ser.pro文件

server-port 游戏端口 改成商户给你的开放端口

如果是转发则改成**被转发端口** 然后用**转发端口**进入游戏

online-mode 是否开启正版

3.疯狂回车出现以下文本则开启成功
