## 非生物因素

### steamcmd

```cmd
steamcmd +login anonymous +app_update 2857200 +quit
```

### 启动命令

steamapps\common\Abiotic Factor Dedicated Server\AbioticFactor\Binaries\Win64

```cmd
AbioticFactorServer-Win64-Shipping.exe -log -newconsole -useperfthreads -NoAsyncLoadingThread -MaxServerPlayers=6 -PORT=7777 -QueryPort=27015 -ServerPassword=123123 -SteamServerName=YunMoan
```

### 游戏配置

steamapps\common\Abiotic Factor Dedicated Server\AbioticFactor\Saved\Config\WindowsServer\GameUserSettings.ini

```xml
[SandboxSettings]

;如果打开，资源将自动在设施周围重生。但是不会刷新家具。(默认:False,开:True,关:False)
LootRespawnEnabled=True

; 默认情况下，设施中的电源在夜间关闭，包括电源插座。(默认:True,开:True,关:False)
PowerSocketsOffAtNight=True

; 此设置将允许禁用昼夜循环。时间将继续像往常一样流逝，但不会有从白天到黑夜或黑夜到白天的过渡。由于有些事情只在白天或晚上发生，这可能会导致不可预见的后果，例如无法获得某些物品。(0:正常,1:总是白天,2:总是晚上)
DayNightCycleState=0

;昼夜循环速度，数字更高，时间过得更快。这不会影响其他冷却时间的速度，重生时间等等。(默认值:1.0,最小值:0.1,最大值:3.0)
DayNightCycleSpeedMultiplier=1.0

; 水槽补充速率，数值越高水槽补充的越快，相反越低就越慢，设置为0的话就禁用。(默认值:1.0,最小值:0.0,最大值:10.0)
SinkRefillRate=2.0

; 食物变质速率，数值越高腐烂的越快，相反越低就越慢，设置为0的话就不会腐烂。(默认值:1.0,最小值:0.0,最大值:2.0)
FoodSpoilSpeedMultiplier=0.1

; 冷藏物品效率，数值越高就越不会变质。(默认值:1.0,最小值:0.0,最大值:2.0)
RefrigerationEffectivenessMultiplier=1.9

; 敌人刷新速率，这个值决定了敌人刷新的频率。数值越低敌人刷新的时间越长，数值越高刷新的越快。(默认值:1.0,最小值:0.5,最大值:3.0)
EnemySpawnRate=1.0

; 敌人血量，越低血越少，越高血越多。(默认值:1.0,最小值:0.75,最大值:3.0)
EnemyHealthMultiplier=1.0

; 敌人对玩家的伤害倍率，越低伤害越少，越高伤害越高。(默认值:1.0,最小值:0.25,最大值:3.0)
EnemyPlayerDamageMultiplier=1.0

; 敌人对部署物的伤害倍率，越低伤害越少，越高伤害越高。(设置为0可能会卡住，默认值:1.0,最小值:0.1,最大值:5.0)
EnemyDeployableDamageMultiplier=1.0

; 敌人警觉玩家的速度，越低越不容易警觉。(默认值:1.0,最小值:0.1,最大值:3.0)
DetectionSpeedMultiplier=1.0

;友伤，这个值将决定玩家对其他玩家造成的伤害，作为正常伤害的比例。如果一件武器造成10点伤害，10%的数值意味着玩家会受到1点伤害。如果想要正常PVP，数值需要调到1以上。(默认值:0.5,最小值:0.0,最大值:3.0)
DamageToAlliesMultiplier=0.0

; 饥饿值消耗速度，数值越高，饿的越快。(默认值:1.0,最小值:0.0,最大值:2.0)
HungerSpeedMultiplier=1.0

; 口渴值消耗速度，数值越高，渴的越快。(默认值:1.0,最小值:0.0,最大值:2.0)
ThirstSpeedMultiplier=1.0

; 疲劳值消耗速度，数值越高，累的越快。(默认值:1.0,最小值:0.0,最大值:2.0)
FatigueSpeedMultiplier=1.0

; 如厕值消耗速度，数值越高，拉的越快。(默认值:1.0,最小值:0.0,最大值:2.0)
ContinenceSpeedMultiplier=1.0

; 经验获取速率，数值越高，获取越多。(默认值:1.0,最小值:0.0,最大值:3.0)
PlayerXPGainMultiplier=1.0

; 物品堆叠数量倍率。(默认值:1.0,最小值:1.0,最大值:30.0)
ItemStackSizeMultiplier=2.0

; 物品重量倍率，数值越高越重。(默认值:1.0,最小值:0.0,最大值:5.0)
ItemWeightMultiplier=0.8

; 物品耐久度倍率，越高耐久越久。(默认值:1.0,最小值:0.0,最大值:5.0)
ItemDurabilityMultiplier=2.0

; 玩家身上的武器和物品，在重生时会损失的耐久度。(默认值:0.1,最小值:0.0,最大值:1.0)
DurabilityLossOnDeathMultiplier=0.0

; 在聊天栏显示死亡信息。(默认:True,开:True,关:False)
ShowDeathMessages=True

; 是否允许分享配方。(默认:True,开:True,关:False)
AllowRecipeSharing=True

; 是否允许使用寻呼机。(默认:True,开:True,关:False)
AllowPagers=True

; 是否允许玩家幻化外观。(默认:True,开:True,关:False)
AllowTransmog=True

; 是否关闭掉配方小游戏，如果为True将会关闭配方小游戏，立刻解锁配方。(默认:False,开:True,关:False)
DisableResearchMinigame=False

; 死亡惩罚。(默认值:1，0:保留所有东西，1:保留装备的物品和快捷栏，2:仅保留快捷栏，3:仅保留装备的物品，4:丢失包含快捷栏在内的所有物品，5:所有物品都被销毁)
DeathPenalties=0

; 全体共享配方解锁进度。(默认:False,开:True,关:False)
GlobalRecipeUnlocks=False

; 开局携带的初始武器。(默认值:0，0:什么都不带，1:短刀，2:桌腿，3:厨刀，4:新手警棍，5:管道棒，6:随机)
FirstTimeStartingWeapon=5

```
