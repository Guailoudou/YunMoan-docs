## killingfloor2-杀戮空间2

### steamcmd

```cmd
steamcmd +login anonymous +app_update 232130 validate +quit
```

### 教程

- 服务端下载好后直接运行直接双击运行 KF2Server即可,这个时候服务器开启了,但我们先关掉,要修改下参数

- 修改服务器自带的网页管理系统路径steamcmd/steamapps/common/killingfloor2/kfgame/Config
  打开KFWeb.ini
  找到bEnabled=
  bEnabled=?
  设置是否打开远程网页管理True（开启）False（关闭）。只可以使用True（开启）False（关闭）来作为结尾。当然，WebAdmin默认是关闭的，如图所示。要开启只需要把默认的false改成true即可。



- 修改服务器基本文件的参数
  路径steamcmd/steamapps/common/killingfloor2/kfgame/Config
  打开PCServer-KFGame.ini
  这个文件涉及到的参数很多，也是服务器最基本，最重要的参数。包括了，服务器难度，服务器波数，服务器密码，管理员密码，服务器名字，地图循环设置【所谓的地图循环，就是一局打完后地图的投票列表。我们设置的就是这个投票列表。当然，在WebAdmin可以更方便的设置，这边也是一种方法，但是更麻烦一点。在受到一些插件冲突导致WebAdmin打不开的时候，就需要在这边进行设置地图循环。】
  那么我现在一个一个的来说。

​	GameDifficulty=0
​	对游戏难度进行设置 =0（普通）=1（困难）=2（自虐）=3（地狱）

​	AdminPassword=
​	设置WebAdmin登录，服务器黄名管理员登陆的密码【可用于开启作弊码，等等】
​	密码默认是空白，请直接在等号后面加上自己想要的密码。

​	GamePassword=
​	设置服务器房间密码（不支持中文）

​	GameLength=2
​	对游戏波数进行设置 =0（4波）=1（7波）=2（10波）
​	服务器默认是7波

​	ServerName=
​	房间名字修改

​	bUseMapList=
​	开启将使用你自定义的地图循环列表，而不是官方默认的。开启（True）关闭（False）。只可以使用True（开启）False（关闭）来	作为结尾。官方默认是开启（True），所以这项其实你完全不用动。知道有个这玩意就可以了。

​	GameMapCycles=
​	地图循环的总代码，用于搜索位置。
​	官方的地图循环是这样的，
GameMapCycles=(Maps=("KF-BurningParis","KF-Bioticslab","KF-Outpost","KF-VolterManor","KF-Catacombs","KF-EvacuationPoint","KF-Farmhouse","KF-BlackForest","KF-Prison","KF-ContainmentStation","KF-HostileGrounds","KF-InfernalRealm","KF-ZedLanding"))
​	可以看得出，都是官图。



- 添加创意工坊的地图或者创建
  路径steamcmd/steamapps/common/killingfloor2/kfgame/Config
  打开PCServer-KFGame.ini

​	[OnlineSubsystemSteamworks.KFWorkshopSteamworks]
​	ServerSubscribedWorkshopItems=
​	在最下面添加这段代码
​	=指的是创意工坊插件或者地图的ID
​	例如:ServerSubscribedWorkshopItems=78984658

- 关掉黑客远程接管你的服务器功能
  路径steamcmd/steamapps/common/killingfloor2/kfgame/Config
  打开PCServer-KFEngine.ini
  使用搜索功能，直接搜索查找 bUsedForTakeover
  你会看到官方默认是 true 开启模式下的，
  将其设置为 false 即可。
  (注意:必须修改)

- 关于杀戮空间2的服务器端口修改(除非开多服,不然没必要修改)路径steamcmd/steamapps/common/killingfloor2/kfgame/Config
  PCServer-KFEngine.ini 两处端口可修改
  port=
  PeerPort=

​	KFWeb.ini 一处端口可修改
​	ListenPort=

- 关于杀戮空间的服务器公告修改路径steamcmd/steamapps/common/killingfloor2/kfgame/Config
  打开PCServer-KFGame.ini
  搜索
  BannerLink=这个是公告图片地址
  ServerMOTD=这个是公告文字内容
  WebsiteLink=自定义的链接,
