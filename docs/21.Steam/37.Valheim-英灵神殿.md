## Valheim-英灵神殿

### steamcmd

```cmd
steamcmd +login anonymous +app_update 896660 validate +quit
```

### 安装路径

```
steamcmd\steamapps\common\Valheim dedicated server
```

### 启动命令

```
D:\steamcmd\steamapps\common\Valheim dedicated server\start_headless_server.bat
```

```cmd
@Echo off
set SteamAppId=892970

echo "Starting server PRESS CTRL-C to exit"

REM Tip: Make a local copy of this script to avoid it being overwritten by steam.
REM NOTE: Minimum password length is 5 characters & Password cant be in the server name.
REM NOTE: You need to make sure the ports 2456-2458 is being forwarded to your server through your local router & firewall.
REM提示：制作此脚本的本地副本，以避免它被steam覆盖。
REM注意：最小密码长度为5个字符，密码不能在服务器名称中。
REM注意：您需要确保端口2456-2458通过本地路由器和防火墙转发到您的服务器。
valheim_server -nographics -batchmode -name "My server" -port 2456 -world "Dedicated" -password "secret"
```

- name - > 服务器名称
  - 可以中文名 但要以**UTF-8**格式保存
  - 甚至支持HTML标签 可以加粗 可以颜色
    - <color=red>服务器名称</color>
- port ->游戏端口 
  - 经测试 这里提供什么 便是延后+1+2 一共三个端口 均要开放
  - 但直连端口是 +1
- world->存档名称
- password ->密码

也可以在游戏目录下

```cmd
.\valheim_server -nographics -batchmode -name "YeziMoan-墨安安安" -port 301 -world "Dedicated" -password "ceshi"
```
