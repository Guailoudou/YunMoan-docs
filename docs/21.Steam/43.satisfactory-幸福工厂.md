## satisfactory-幸福工厂

### 官方开服

https://satisfactory.wiki.gg/wiki/Dedicated_servers

### steamcmd

```cmd
steamcmd +login anonymous +app_update 1690800 validate +quit
```

### 安装目录

```
steamcmd\steamapps\common\SatisfactoryDedicatedServer
```

### 配置目录

```
C:\Users\<用户名>\AppData\Local\FactoryGame\Saved\Config\WindowsServer\Game.ini
```

```ini
[/Script/Engine.GameSession]
MaxPlayers=32
ServerName=我的幸福工厂服务器
ServerPassword=服务器密码
```

## 启动命令

```bash
FactoryServer.exe -Port=301 -ReliablePort==302 -ExternalReliablePort=301 -ini:Game:[/Script/Engine.GameSession]:MaxPlayers=8
```

- **port UDP 游戏端口** **必须**
  - **可映射转发**
- ReliablePort TCP 
  - 覆盖服务器使用的可靠消息传递端口。默认端口为 8888（TCP）。如果指定的端口已被占用，服务器将无法初始化。
- ExternalReliablePort
  - 如果使用外部端口重映射， 请同时使用此选项。
