# Dayz

### steamcmd

```
steamcmd +login 账户 密码 +app_update 223350 validate +quit
```

> [!NOTE]
>
> dayz 不允许匿名登录

### 启动命令

```cmd
@echo off
cls

set version=1.0
set wat=Dayz SA


title %wat% Watchdog


cd D:\steamcmd\steamapps\common\DayZServer

:watchdog
echo (%time%) %wat% started.
start "Dayz_SA" /wait /affinity FF /high "DayZServer_x64.exe" -config=serverDZ.cfg
echo (%time%) %wat% closed or crashed, restarting.
goto watchdog
```

带自动重启

- -config=serverDZ.cfg (服务端基本配置文件)

- -port=2302 (服务端端口)

- -profiles=D:\steamcmd\steamapps\common\DayZServer (服务端设置路径)

- -freezecheck (冻结超过5分钟后停止服务器并创建转储文件)

- -noFilePatching (确保只加载PBO，没有解包数据。)

启动参数限制命令:

- -cpucount=X (指你的CPU核数)
- -exThreads=7（是指对你的CPU的线程使用率8核设置为7，4核以上CPU设置3~5，双核1~3）
