## 百慕大野兽

### steamcmd

```cmd
steamcmd +login anonymous +app_update 882430 validate +quit
```

### 启动命令

steamcmd\steamapps\common\Beasts of Bermuda Dedicated Server\WindowsServer

```cmd
start BeastsOfBermudaServer.exe -GameMode Life_Cycle -MapName Forest_Island -SessionName 服务器名称 -NumPlayers 32 -MultiHome=0.0.0.0 -Port=7777 -QueryPort=27015 -log
```

> [!NOTE]
>
> 服务器名称应该只能英文，这里只是提示

```ini
GameMode=Life_Cycle
选择Life_Cycle生命周期、Combat战斗或Free_Roam自由漫游。
这将是服务器上使用的模式。

GrowthLimit=100.0f
这是此服务器上的增长限制。
请注意，任何超过2的值都不太可能被玩家获得，除非他们献身于此。
增长是逆指数的，超过1.2会变得非常缓慢。

bConsoleLocked=true
如果设置为false，任何人都可以访问控制台，因为服务器将处于打开控制台模式，将每个人都视为默认管理员。
除非您正在进行测试，否则不建议将其设置为false

ReservedAdminSlots=2
这是为服务器上的管理员保留插槽人数位子的设置。
当管理员加入游戏时，这些插槽人数位子将被填满。其默认值为2。

ServerAdmins=(UserSteamId64=76561198003843972,AdminRank="DefaultAdmin")
这表示服务器管理员的条目。在百慕大野兽中，服务器支持管理员等级，您可以为服务器设置自定义管理等级。
请注意，这被视为一个列表。为每个管理员添加新的ServerAdmins=（…）行。

AdminRanks=(AdminRank="DefaultAdmin", RankLevel = 0)
这表示具有关联级别的管理员级别。级别越高，管理员的级别越高。
任何比其他管理员级别更高的管理员都可以做其他管理员可以做的任何事情，等等。

AdminCommandRules=(CommandType = EAdminCommands::TheAdminCommand, MinimalRank="RankName")
这是管理命令的规则。每个管理员命令类型由EAdminCommands表示。一个值可以访问多个命令。
最小值是使用此命令所需的最小值。

SteamGroupName="Name"
如果您的服务器在steam上有一个组，您可以将其名称放在这里，玩家可以通过游戏界面找到并加入您的组。

bRequiresSteamGroupToJoin = false
您的服务器是否需要加入使用“SteamGroupName=”Name“指定的steamgroup。
不属于此组的玩家将被拒绝访问您的服务器。

DiscordLink="GZrhgrC"
如果你有Discord。玩家可以通过游戏界面找到它。
出于安全原因，链接必须仅包含与邀请相关联的密钥。

bDisplayIntroMessage=false
是否显示欢迎消息。有关如何设置欢迎消息的详细信息，
请参阅Beasts of Bermuda百慕大野兽每日消息设置教程。

'bUseHardGroupLimits=false
这对组设置了硬限制。超过CreatureLimits中定义的值时，具有组限制的生物将不允许在一起。
如果保持为false，则游戏将对试图分组超过限制的玩家带来指数级增长的不适感。

bDisableGlobalChat=false
是否禁用服务器上的全局聊天。本地和群聊仍将可用。

!CreatureLimits=ClearArray
+CreatureLimits=(CreatureType=EDinoType::TheDinoType, PercentAllowed=100.0f, GroupLimit=999, bRequiresVeteran=False)

!PlayerChatColors=ClearArray
PlayerChatColor=（SteamID=PlayerId64，ChatColor=（R=1.00，G=0.00，B=0.0，A=1.0））
这些名称颜色覆盖管理员黄色聊天颜色
他们也会出现在本地、聚会和消息聊天中

!PlayerChatTags=ClearArray
PlayerChatTags=(SteamID=PlayerId64,ChatTag="admin")
这个是给管理员添加前缀.可以写中文
他们也会出现在本地、聚会和消息聊天中

之后，您需要为自动保存配置创建一个新部分。它必须位于文件的末尾，标题为：
[/Script/BeastsOfBermuda.SaveSystem]
AutosaveTime=300.0f
服务器将自动保护其数据并将其写入磁盘的时间（以秒为单位）。该定时器被限制为最小30秒，默认为120秒。
```
