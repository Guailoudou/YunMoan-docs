## Unterned-未转变者

### 官方文档

https://docs.smartlydressedgames.com/en/stable/servers/server-hosting.html

### steamcmd

```cmd
steamcmd +login anonymous +app_update 1110390 validate +quit
```

### 启动命令

steamapps\common\U3DS

```cmd
start "" "%~dp0ServerHelper.bat" +LanServer/Example
```

#### 可添加参数

游戏端口

-Port 27015

> [!NOTE]
>
> 虽然只写一个 但实际占用三个 27015 27016 27017
>
> 27015 实际链接使用
>
> 27016 steam服务器列表
>
> 27017 steam后端

局域网网络/服务器存档目录名字

+LanServer/Example

互联网网络/服务器存档目录名字

+InternetServer/My Server
服务器名字:
-Name My Server
地图名字:
-map Washington

服务器密码:
-Password 123456

视角: 指定服务器玩家第一人称视角和第三人称视角
-Perspective "First","Third","Both","Vehicle"

允许管理员调用作弊命令,如从聊天中生成物品或车辆.
-Cheats
