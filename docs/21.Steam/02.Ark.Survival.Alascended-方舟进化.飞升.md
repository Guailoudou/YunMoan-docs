## Ark.Survival.Alascended-方舟进化.飞升

### steamcmd命令

```cmd
steamcmd.exe +login anonymous +app_update 2430930 validate +quit
```

### 启动命令

```cmd
start YOUR_ARK_SERVER\ShooterGame\Binaries\Win64\ArkAscendedServer.exe TheIsland_WP?listen?SessionName="SERVERNAME"?Port=7777?QueryPort=27015?-WinLiveMaxPlayers=70 -UseBattlEye
```

- TheIsland_WP 地图默认
- listen 监听
- SessionName 服务器名称
- Port 游戏端口
- QueryPort 查询端口
- WinLiveMaxPlayers 最大游戏人数
- mods 模组Id多个使用逗号分割
